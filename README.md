# Gemini Flask Chatbot

A simple chatbot built using Flask and the Google Gemini API.

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Add your API key

Edit the `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

## Run

```bash
py app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
